# Festival_des_jeux

Festival_des_jeux est le front d'une application mobile développée en Swift qui utilise le framework SwiftUI.

## Installation

Vous retrouverez le back de cette application sur le git à l'adresse : (https://github.com/tiffany-dumaire/Backend-Festival-App)
Vous pouvez récupérer également le code du front de l'application mobile sur le git à l'adresse : (https://github.com/tiffany-dumaire/Festival_des_jeux)

## Hébergement

Le back de l'application a été hébergé sur Heroku. 
Vous pourrez donc également retrouver ce code hébergé à l'adresse url suivante : (https://backend-festival-app.herokuapp.com/)

## Exécution

Après avoir récupérer le code du Front de cette application. 
Vous pourrez le lancer à l'aide du logiciel xcode sur un mac ou un mac à distance.
